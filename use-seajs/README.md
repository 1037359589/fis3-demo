## use-seajs

http://seajs.org

基于 seaJS 的模块化解决方案

### 注意

- 文件 md5 戳功能
- 支持 all in one 打包。但是：不支持部分打包！！

## 插件安装

```bash
$ npm instll fis3 -g
$ npm install [-g] fis3-hook-module
$ npm instlal [-g] fis3-postpackager-loader
```

## 编译 & 预览

```bash
$ fis3 release
$ fis3 server start
```

## 产出产品代码

```bash
$ fis3 release prod -d ./output
```
